GitHub Star Kit

AI-powered intelligence for your GitHub stars — weekly digest, semantic search, smart tags, code-context search. All local.

View on GitHub Privacy Policy
🔒 Fully local-first. No telemetry. No servers. No analytics. The only network destinations are api.github.com and the AI provider you choose. Your data stays on your device.

Key features

📰 Weekly Digest

Top recently-pushed starred repos ranked by relevance to your interest profile. AI summaries explain why each matters.

🔍 Semantic Search

Type "SSE retry library" and find the repo even if its description never said those words. Local vector search.

🤖 Auto-Tags

One-click LLM tagging across your whole star list. Filter the manage page by tag.

🌐 Translate (11 langs)

Repo descriptions + tags translated to zh-CN / ja / ko / de / fr / es / pt-BR / ru / vi / zh-TW.

🔧 Deep-Index Code

Opt-in: index source code from selected repos. Search code chunks with file path + GitHub permalink.

📝 Private Notes

Add a private memo to any starred repo. Stays on your device — GitHub never sees it.

Bring your own AI key

Star Kit doesn't ship with an API key. You provide one for the provider you trust: OpenAI, Anthropic, Voyage AI, SiliconFlow, DashScope, or a self-hosted Ollama proxy on localhost.

Typical cost on the cheapest providers: ~$0.05-$0.50 to index 1,000 stars; ongoing sync is free.
